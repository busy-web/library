@busy-web/utils
==============================================================================

Installation
------------------------------------------------------------------------------

Install this addon using ember-cli
```
ember install @busy-web/utils
```

Usage
------------------------------------------------------------------------------

Use @busy-web/utils in your js files
```
import { Assert } from '@busy-web/utils';

Assert.test('some message', someTest === someValue);
```
